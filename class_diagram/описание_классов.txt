package game:
1) Launch - класс точки входа
2) DisplayObject - класс для отображения нужных графических элементов, задание ширины и высоты
3) Keyboard - класс для обработки и обновления ключей с клавиатуры
4) Game - главный класс самой игры, создание игрового потока
5) GameField - класс для работы с игровым полем
6) Cube - класс для работы с кубиками на поле (отрисовка, анимация, поведение)
7) Leaders - класс для обработки и хранения лучших результатов игры
8) Scores - класс для обработки и хранения общих результатов игры
9) SetAudio - класс для работы со звуковыми эффектами
10) Spot - класс для задания строк и столбцов по координатам в игровом поле

package gui:
1) Screen - главный класс гуи: отвечает за работу кликов и за активацию определеных панелей
2) PanelMenu - класс для работы с панелью главного меню
3) PanelGame - класс для работы с панелью самой игры
4) PanelScores - класс для работы с панелью счета игры
5) PanelButton - класс для работы с кнопками
6) Button - класс для создания кнопок
